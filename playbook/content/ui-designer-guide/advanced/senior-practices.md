# Senior UI Designer: Advanced Design Systems & AI Tools

## Pattern 1: AI-Assisted Design Exploration
Use Midjourney, DALL-E for visual design exploration and mood boards.

## Pattern 2: Design System Automation
Leverage Figma AI plugins to maintain consistency and generate component variations.

## Pattern 3: Accessibility-First Design
Use AI tools to check color contrast, WCAG compliance, and inclusive design patterns.
