# AI Onboarding Prompts

## Purpose
Use these prompts to help new team members understand the codebase, architecture, and standards quickly.

## Prompts

### Codebase Exploration
```
Explain the architecture of this project based on the folder structure and README files.
Identify key components and their relationships.
```

### Understanding Complex Logic
```
Explain this function/class step-by-step in simple terms.
What are the inputs, outputs, and side effects?
Create a sequence diagram description for this flow.
```

### Setup Assistance
```
Guide me through the local environment setup for this project.
Based on package.json, what are the key dependencies and scripts?
How do I run the tests and build the project?
```
