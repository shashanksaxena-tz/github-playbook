# AI ROI Calculator

## Inputs

| Parameter | Value | Description |
|-----------|-------|-------------|
| **Team Size** | 10 | Number of developers |
| **Hourly Rate** | $100 | Average fully loaded hourly cost |
| **Tool Cost** | $19 | Monthly cost per user (e.g., Copilot) |
| **Productivity Gain** | 10% | Estimated efficiency improvement |

## Calculations

### Monthly Cost
`Team Size * Tool Cost` = 10 * $19 = **$190 / month**

### Monthly Savings
`Team Size * (Hourly Rate * 160 hours) * Productivity Gain`
10 * ($100 * 160) * 0.10 = **$16,000 / month**

### Net ROI
`(Savings - Cost) / Cost * 100`
($16,000 - $190) / $190 * 100 = **8,321%**

## Break-Even Analysis
Time saved required to cover cost:
`Tool Cost / Hourly Rate` = $19 / $100 = **0.19 hours (11.4 minutes) per month**

*If a developer saves just ~12 minutes per month, the tool pays for itself.*
