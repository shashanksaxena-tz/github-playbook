# AI-Assisted Onboarding Checklist

## Phase 1: Access & Setup
- [ ] Install IDE (VS Code recommended)
- [ ] Install GitHub Copilot extension
- [ ] Authenticate with GitHub account
- [ ] Configure AI tool settings (inline suggestions, chat)

## Phase 2: Training
- [ ] Complete "Getting Started with GitHub Copilot" guide
- [ ] Review team prompt library
- [ ] Practice "Comment-Driven Development"
- [ ] Learn context management techniques

## Phase 3: Practice
- [ ] Pair program with AI on a simple bug fix
- [ ] Use AI to explain a complex module
- [ ] Write unit tests using AI generation
- [ ] Refactor a legacy function with AI assistance

## Phase 4: Review
- [ ] Submit first AI-assisted PR
- [ ] Review AI usage guidelines
- [ ] Check understanding of security policies
